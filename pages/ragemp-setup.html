RAGE:MP SETUP GUIDE

Follow this guide carefully to avoid common setup issues. Prime Roleplay only supports the official RAGE:MP client.

STEP 1 — OWN GTA V (ORIGINAL)

You must own a legitimate copy of GTA V from one of the following:

  • Steam
  • Epic Games
  • Rockstar Games Launcher

Cracked or modified versions are not supported.

STEP 2 — DOWNLOAD RAGE:MP

Download RAGE:MP only from the official website. Do not use third-party mirrors.

DOWNLOAD RAGE:MP (OFFICIAL)

Supported on Windows 10 / 11 (64-bit)

STEP 3 — LINK GTA V TO RAGE:MP

After installing RAGE:MP:

  1. Launch RAGE:MP
  2. Select your GTA V installation folder
  3. Wait for verification to complete

This step is required only once.

STEP 4 — INSTALL Prime RP LAUNCHER

Download and install the Prime Roleplay Launcher. Run it as administrator on first launch.

The launcher handles updates and server connection automatically.

COMMON ISSUES

Launcher not detecting GTA V?

Make sure GTA V is launched at least once.

RAGE:MP crashes on launch?

Disable overlays and antivirus temporarily.

Server not visible?

Check that you are using the latest RAGE:MP version.
